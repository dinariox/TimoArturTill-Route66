Wenn Ihnen der Besuch bei uns gefallen hat, oder wenn Sie Vorschläge haben, können Sie uns hier Feedback geben

Wir freuen uns, wenn Sie uns positiv bewerten, aber lesen uns auch gerne Kritik und Verbesserungsvorschläge an!

Falls Sie eine private Nachricht an uns verfassen möchten, können Sie das unter Kontak machen.

Hier können Sie sich Feedback und Bewertungen von anderen Gästen durchlesen. Möchten Sie auch Ihre Meinung sagen? Klicken Sie dazu einfach auf Feedback geben. Sie werden dann auf ein Formular weitergeleitet, in dem Sie Ihre Nachricht so wie eine Wertung von 1 bis 5 Sternen abgeben können. Wenn Sie möchten, können Sie auch Ihren Namen angeben. Ansonsten wird der Beitrag anonym gepostet.
